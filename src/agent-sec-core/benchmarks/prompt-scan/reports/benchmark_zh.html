评测数据集

背景：中文 prompt injection / 越狱攻击领域的公开数据集较为匙乏。大多数“中文 LLM 安全数据集”（如 JADE、FLAMES、SafetyBench 等）实际上是内容安全类——测试模型是否输出有害内容，而非提示词攻击类——测试攻击者能否操控系统行为。本基准整合了当前公开可用的、真正针对 prompt injection / 越狱攻击手法的中文数据集，并自建良性样本用于误报率测试。数据集总计 480 条（攻击 343 条 + 良性 137 条），攻击：良性 ≈ 2.5:1，达到业界评测基准的推荐水平。
Alibaba-AAIG / Strata-Sword
★★★
200 条 Apache-2.0
阿里巴巴安全团队出品，目前唯一系统性覆盖中文特有攻击手法的公开数据集。分层越狱评测集 Level 2+3，含拆字、藏头诗、灯谜等其他数据集均无覆盖的中文特有攻击向量。
CPAD
★★
70 条 Apache-2.0
学术论文配套数据集（AAAI 2024），公开可用的唯一针对中文攻击模板的数据集。覆盖赋予角色、加壳、对话补全等攻击模板。
THU-COAI / Safety-Prompts
★★★
25 条 MIT
清华大学 CoAI 实验室出品，发表于 ACL 2023，学术引用广泛。Goal_Hijacking + Role_Play 攻击，当前基准中召回率最高（72%）。
Meta-PromptGuard-Multilingual
★★★★
28 条 CC-BY-4.0
Meta 官方评测集，是 Llama-Prompt-Guard-2 模型的官方配套基准，行业参考标准。覆盖直接注入与间接注入的多语言中文部分。
BIPIA-adapted-zh
★★★
20 条 MIT
基于 Microsoft Research 的 BIPIA 基准改编，BIPIA 是间接注入领域的标准学术基准。中文适配版本，覆盖信息注入、诈骗注入、编码混淡、代码注入等。
Self-Constructed（自建）
137 条良性 internal
内部构建的良性样本，用于误报率测试。包含日常问答（BN-001~BN-112）、边界良性（ADV-7）、含注入载荷片段的正当请求（MIX-3）、角色扮演良性（EXT-RP-015）。

⭐⭐⭐
Recall (TPR) · 首要指标
攻击样本中被检出的比例（TP / (TP+FN)）。漏检攻击的代价最高，是安全场景下最重要的单一指标。
⭐⭐⭐
F1 Score · 核心指标
Precision 和 Recall 的调和均值。综合衡量检测精度，平衡精确率与召回率。
⭐⭐
Precision · 精确率
检出结果中真正攻击的占比（TP / (TP+FP)）。FP 过高会降低用户体验，但误报代价低于漏检。
⭐⭐
Balanced Accuracy
(TPR + TNR) / 2。对不平衡数据集更公平，补充 Accuracy 在 2.5:1 不平衡场景下的缺陷。
⭐
Accuracy · 准确率
整体准确率（(TP+TN) / total）。参考意义有限：在 2.5:1 不平衡下，全预测攻击可达 72% Accuracy。
